# **App Name**: Invoice Recovery CRM

## Core Features:

- Client Management: Add, update, and manage client information including name, company, contact details, etc.
- Invoice Case Creation: Create new invoice recovery cases, linking them to existing clients.
- Case List and Filtering: Display a list of all recovery cases, with filtering by status and sorting by due date.
- Case Detail View: View all details for a specific case and update the status and follow-up notes.
- API Endpoints for Clients: Provide RESTful API endpoints for client management (POST, GET).
- API Endpoints for Cases: Provide RESTful API endpoints for case management (POST, GET, PATCH).
- Database Schema: Maintain MySQL/PostgreSQL relational tables for clients and cases, ensuring proper relationships and constraints.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to convey trust and professionalism.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#008080) to highlight key actions and statuses.
- Headline font: 'PT Sans', sans-serif, for headlines; body font: 'Inter', sans-serif, for body text, since PT Sans does not perform well in long form.
- Use simple, professional icons for statuses and actions within the CRM.
- Maintain a clean and organized layout, focusing on ease of navigation and quick access to key information.
- Subtle transitions and animations to enhance user experience without being distracting.
